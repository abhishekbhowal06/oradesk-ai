# DENTACOR OS

**AI-Powered Dental Front-Desk OS**

## Project info

Enterprise-grade AI automation for dental clinics. Automate bookings, calls, and follow-ups with clinical precision.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting Started

1. Install dependencies:
```sh
npm install
```

2. Start the development server:
```sh
npm run dev
```

3. Build for production:
```sh
npm run build
```
